fun String.isspace(): Boolean
Content copied to clipboard

Return true if there are only whitespace characters in the string and there is at least one character, false otherwise. A character is whitespace if in the Unicode character database, either its general category is Zs (“Separator, space”), or its bidirectional class is one of WS, B, or S.
